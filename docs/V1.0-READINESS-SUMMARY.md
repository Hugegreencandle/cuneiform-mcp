## Methods paper claims (57)

### §1-§3.4: foundations (claims 1-13)
Lexical/fuzzy/thematic/damaged/scribal axes, calibration-limited precision, run-bonus calibration methodology.

### §3.5-§3.14: discoveries and methodology (claims 14-32)
Lacuna restoration 92% top-1 (parallel-template aligned), bi-orphan surface methodology, BM.77056 *āšipūtu* cluster (KAR-44 curriculum recovery), sub-tablet chunk-parallel discovery, sign2vec embeddings + multiple variants, per-period diachronic isolation.

### §3.15-§3.17: cross-axis ML (claims 33-37)
Chunk-hash partition (NA/NB), Bayesian fusion bootstrap (n=12), joins graph (4,361 fragments), numerical filter, sign2vec-augmented lacuna restoration (90% disagreement between α=0 / α=1).

### §3.18-§3.22: composition tools (claims 38-42)
Active-learning feedback loop, composition assignment via registry, rooted stemma (Mīs pî has NO pre-NA witness in cluster), fragment completeness (sign_count vs chunk_coverage dissociation), composition lineage (Mīs pî single-provenance Kuyunjik).

### §3.23-§3.25: damaged passages + calibration (claims 43-45)
Probabilistic damaged-passage classification with restoration marginalization, panel review (15 asks, 12 shipped + 3 gated), single-sign restoration ECE 0.6490 (overconfident).

### §3.26-§3.28: data layers (claims 46-48)
ABZ glyph lookup (OGSL ∩ eBL), citation network (71 scholars, 7 bridge), lemma-Jaccard parallel (complementary to sign-trigram).

### §3.29-§3.31: archaeology + calibration closure (claims 49-51)
Ancient find-spot clustering (Kuyunjik=15,604 tablets across 9 prefixes), cross-axis disagreement (K.5896 lemma_silent), Platt-calibrated lacuna scores (ECE 0.6374 → 0.0109, 58× improvement).

### §3.32-§3.33: methodology (claims 52-53)
Held-out evaluation methodology (test AUC 0.67 at n=10 initial pass), active-learning target recommendation (chunk-overlap percentile proxy).

### §3.34-§3.37: NEW since v0.53 (claims 54-57)
- **§3.34 / Claim 54 (v0.54):** Corpus-wide composition-assignment discovery — 310 candidate exemplars at p>0.9 surfaced from a 57-second scan across 36,498 transliterated fragments.
- **§3.35 / Claim 55 (v0.55):** `list_candidate_exemplars` tool exposes discovered exemplars; tool count crosses 100 milestone.
- **§3.36 / Claim 56 (v0.56):** `composition_assignment_match` added as 6th feature to v0.29 Bayesian fusion. Held-out test AUC 0.667 → **1.000** at n=42. Lemma index 216→504 tablets enrichment burst. ABZ→Unicode MZL fallback recovers 4 hard fails (𒄰 𒄒 𒊻 𒉚).
- **§3.37 / Claim 57 (v0.57):** Platt scaling applied to v0.29 logistic regression = HONEST NULL RESULT. Pre-Platt ECE 0.0165 (already well-calibrated); post-Platt 0.0140 (marginal); accuracy unchanged 0.9524. v0.29 is calibration-disciplined out of the box. Contrast with v0.30 lacuna fusion (Platt gave 58× lift). **Different model classes warrant different calibration assumptions** — methods-paper contribution distinguishes calibration discipline as model-class-dependent.

- **§3.25**: `restore_lacuna_semantic.joint_score` overconfident 13× ECE threshold (ECE 0.6490 at n=50)

- **§3.36 (NEW)**: 6th feature `composition_assignment_match` lifts held-out AUC to **1.000** at n=42 — separate confirmation that BM.77056↔K.5896 misclassifications were genuine model errors, not calibration artifacts
